Arduino Project (5/28/19)

Project Code

On May 28th, 2019, my friend, Chris and I worked on a quick and simple Arduino Project. It esseentially counts how many times the user clicks a button, and a piezo buzzer will make a sound everytime the button is clicked. Simple as that. We used a 16x2 LCD screen, a piezo buzzer, large button and a Arduino Uno with a small breadboard next to it.

Some of the problems that we encountered were the code not working properly and we switched up the piezo buzzer and the switch since they were malfunctioning. The resolve the code, we first reached out to a friend that has experience with coding C++. Once that was fixed, it was all about picking the right piezo buzzer and button until they actually worked.
