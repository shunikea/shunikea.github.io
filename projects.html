Thunkable Project (9/24/18)

The very first step to my program was to add all of the necessary images and labels in order for my app to be functional and to be able to start coding. Once everything was in place, I started coding. The code is pretty basic, just some variables and functions in order for the timer to actually work. The difficulties that I encountered were the timer not working properly, and the fact that my image bounces up and down every single time I click the blue button. I resolved it by changing the milliseconds of the timer and enabling loops, and I fixed my image glitch by adding a column right above the blue button, and changing the position of it.
